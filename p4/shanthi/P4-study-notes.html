So we will have to create the CSSOM ASAP. Completeing the CSSOM, will unblock the JS Engine so we will be able to execute JS as soon as we have received it. Once the JS is done, we can resume and finish constructing the DOM) 4) Execute JS 5) Merge DOM and CSSOM into the Render Tree 6) Run Layout, Paint page.
